RN 0.43后Text行数过长

发表于 2017-05-19 | 分类于 React Native |

上周升级RN至0.43后，之前用RN写的页面在iOS上竟然挂掉了，Android是OK的。

Text 上有个 numberOfLines 的属性，设置为 0 ，表示多行。当我设置到 400 行时，在iOS手机上已经开始出现渲染问题了。

解决方案：采用 webView 控件。

自己去写html代码，用 webView 的方式加载

scalesPageToFit 这个属性也很有意思，一开始看官方文档没看懂，直到实践才发现，这个值竟然 跟平台有关系。 false 在iOS上字体大小正常，在android上字体缩小，true 则相反。
