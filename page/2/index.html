NSUserDefaults

简介

根据官方文档，翻译一下：类 NSUserDefaults 为与系统之间的交互提供了一个编程接口。系统允许应用程序去定制自己的行为来满足用户的喜好。比如说，你可以允许用户去决定文档自动保存的频率。应用程序通过在 用户默认的数据库中设置value 来记录这些喜好。这些参数在启用应用程序的时候，将被作为默认值。这些偏好设置记录在应用包的plist文件中。

举个例子：app有一个弹红包的功能，只能在用户首次进入app的时候弹出。如何判断 用户是首次进入app呢？我们可以设置一个字段，当app打开时，取出该字段的值，判断是否是首次进入，判断完毕后，将该字段置为false，并存入到 NSUserDefaults。

特点： 适合 存储 轻量级的本地数据

github Pages

首先，你得有个 github 账号。

github Pages 本来是用于介绍托管在github的项目的，可以用来搭建成一个博客。

每个 github 账号只有一个仓库用来存放个人主页，而且仓库的名称必须为 username/username.github.io ，搭建博客完成后，我们可以直接通过 http://username.github.io 访问到个人主页。

在个人的 github 上，创建一个新的 repository ，命名为 username.github.io ，此处的 username 便是你的帐号名称，最后的结果 为 yourname/yourname.github.io。
